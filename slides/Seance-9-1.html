Format et types de format
Structure d'un document numérique
  • Niveau 1 : Vue séquentielle - suite de symboles (caractères)
  • Niveau 2 : Structure arborescente - syntaxique (mots, phrases, paragraphes)
  • Niveau 3 : Vue métadonnées - informations autour du document (date création, auteur, mots-clefs)
  • Niveau 4 (facultatif) : Vue stylisée - présentation (typographie, taille, marges)

Formats textuels
  • .txt : textes bruts, éditeurs de texte
  • .odt : textes enrichis, OpenOffice et autres
  • .doc et .docx : textes enrichis, MicrosoftWord
  • .pdf : textes pour impression, Adobe
  • .html : hypertextes, Web

Déclinaisons de formats selon le type d'informations

Plein Texte / Texte brut
  • Juste du texte
  • Sans formatage, enrichissement, mise en page
  • Vue séquentielle - simple séquence de caractère
  • ASCII : American Standard Code for Information Interchange : encodage des caratères (anglais)
  • Unicode : standard encodage des caractères (autres langues)
