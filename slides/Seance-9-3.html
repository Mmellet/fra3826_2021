Conversion - Do It Yourself
1ere étape : Le Template
    Exemple de commande :
  • pandoc --standalone --bibliography Bibliographie.bib --template=Template.latex -f markdown -t latex TexteDebut.md -o TexteFinal.tex
  • cher pandoc version standalone, le fichier "Bibliographie.bib" est ma bibliographie, mon template de conversion est le fichier "Template.latex", convertis depuis (-f pour from) le Markdown vers (-t pour to) le Latex, mon fichier de départ est "TexteDebut.md", et mon fichier de destination (-o pour out) sera "TexteFinal.tex"
Conversion - Do It Yourself
2ème étape : vers le PDF
    Conversion avec Xelatex
  • xelatex TexteFinal.tex
  • package xelatex convertis vers le format PDF mon fichier "TexteFinal.tex"
